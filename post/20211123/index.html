第三章JavaScript语言基础

JS有六种基本类型，Boolean, Null, Undefined, Number, String, Symbol

Undefined

变量未定义会报错，定义了未初始化则为undefined

Null

逻辑上讲，null为一个空对象指针，所以typeof null返回object

Boolean

Number

0.1+0.2 ==0.3000000000...4
存在一个NaN（非数类型）
NaN涉及的操纵返回NaN
NaN参与的逻辑运算都返回false
数值转换 Number(), parseInt(), parseFloat()
通常使用parseInt()

Object 复杂数据类型

位操作符

按位非~ 取反减一
按位与 &
按位或 |
左移运算符<<n 二进制往左移动n位
右移运算符>>n 二进制往右移动n位
（无符号）<<< >>>

逻辑运算符 && ||

将两项都转换为Boolean类型来计算
&&若第一项为false，则返回第一项，
若第一项为true，则返回第二项

等于和全等于

分别为== 和===
如果使用==，则两个操作数先进行强制类型转换（都转换为Number类型），再去比较是否相等。
如果使用全等===，则不进行强制类型转换

逗号运算符

返回其最后一个值

注意点

逻辑运算符是强制转换为Boolean类型在进行操作
比较运算，等于运算是强制转化为Number类型进行计算。
